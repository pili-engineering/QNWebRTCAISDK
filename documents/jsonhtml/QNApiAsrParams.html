Property
aue
  • aue: number

  • 数据格式，1: pcm，2: AAC，3: MPEG2;默认1

e
  • e: number

  • 请求时间戳, 单位秒

key_words
  • key_words?: string[]

  • 识别关键字; 相同读音时优先识别为关键字。每个词 2-4 个字, 不同词用 `,` 分割

model_type
  • model_type?: number

  • 识别语言，中文: 1, 英文: 2, 中英混合: 0; 默认 1

token
  • token: string

  • 请求签名, [签算方式](https://developer.qiniu.com/kodo/1202/download-token)

voice_id
  • voice_id?: string

  • 数据流id，不同流不同

voice_sample
  • voice_sample: number

  • 数据采样率，取值: 48000, 44100, 32000, 16000, 8000;默认16000
