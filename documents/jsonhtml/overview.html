QNRTCAI 是结合了 QNRTC 和 dora ai 的 SDK，包含语音识别、人脸识别、人脸对比等功能
QNRTCAI

QNRTCAI 提供了一系列的方法，可以通过它获取 SDK 的版本号、语音识别、人脸检测等。

可通过 import * as QNRTCAI from "qnweb-rtc-ai" 导入，然后通过 QNRTCAI.QNAudioToTextAnalyzer 来调用方法。

也可以通过 import { QNAudioToTextAnalyzer } from "qnweb-rtc-ai" 导入，然后直接调用 QNAudioToTextAnalyzer 的方法。

导入方式如下：

import * as QNRTCAI from "qnweb-rtc-ai" 
或者
import { QNAudioToTextAnalyzer } from "qnweb-rtc-ai"
